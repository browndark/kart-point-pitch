# 🎮 Kart Point - Pitch Website

Bem-vindo ao site pitch oficial do **Kart Point** - Um jogo de corrida inovador com mecânicas únicas de coleta e pontuação!

## 📋 Sobre o Projeto

**Kart Point** é um jogo de corrida 2D/3D híbrido onde os jogadores devem:
- 🏎️ Desviar de outros karts pilotos e obstáculos na pista
- 💰 Coletar moedas para acumular pontos
- 🏆 Competir para bater recordes de pontuação
- ⚡ Evitar colisões que resultam em perda de pontos

## 🎯 Mecânicas Principais

### Sistema de Pontuação

**Ganha Points:**
- +10 pontos por moeda coletada
- +50 pontos por desvio perfeito
- +100 pontos por combo de 5x moedas
- +25 pontos por volta completada

**Perde Points:**
- -20 pontos por colisão com obstáculo
- -15 pontos por colisão com outro kart
- -10 pontos por sair da pista
- -50 pontos por capotagem

## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Clone ou abra a pasta do projeto
2. Abra o arquivo `index.html` no seu navegador web

### Opção 2: Com Live Server (VS Code)
1. Instale a extensão "Live Server" no VS Code
2. Clique com botão direito em `index.html`
3. Selecione "Open with Live Server"

### Opção 3: Com Python (Servidor Local)
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Após executar, acesse: `http://localhost:8000`
